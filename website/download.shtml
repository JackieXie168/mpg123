Occasional Win32 and Win64 binaries

Have a look into the download/win32/ or download/win64/ directory for zip files with mpg123 builds (with libmpg123 DLL or static, different decoder options). I don't promise to update these too often. You can always download the source code and compile it with MinGW32 in MSYS.

But you Windows folks are lucky since version 1.6.0: Patrick joined the team as a Windows developer and will provide binaries more regulary! Also, JonY chipped in with cross-compiling builds for 64bit Windows.

Developer Version

Development happens in the subversion repository on orgis.org:

svn://orgis.org/mpg123/trunk

You can also browse it via the ViewVC web interface. You can get snapshots from both places.

A nightly tarball with the the autotool stuff prepared is at http://mpg123.org/mpg123-svn-snapshot.tar.bz2.
